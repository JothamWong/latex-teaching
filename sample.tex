    \section{Code Blocks}

        Using \texttt{minted} package to write code blocks.

        \begin{minted}{python}
            # This is python code
            def factorial(n):
                return n == 1 ? 1 : n * factorial(n-1)
        \end{minted}

        \begin{minted}{java}
            // This is java code
            public int factorial(int n) {
                if (n == 1) {
                    return 1;
                } else {
                    return n * factorial(n-1);
                }
            }
        \end{minted}

        Notice the syntax highlighting. (Basically verbatim + syntax)
